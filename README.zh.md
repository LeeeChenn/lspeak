## ✨ 产品简介

这是一款专为英语学习者打造的 **Mac 英语播放器**，通过 **视频播放 + 字幕切换 + 跟读录音** 等功能，帮助你沉浸式提升英语听说水平。

---

## 🔧 核心功能

- 🎬 **视频播放**：支持导入本地视频，随时随地边看边学；  
- 📝 **字幕切换**：支持中英文字幕自由切换，助你精准理解；  
- 🕒 **变速播放**：语速自由调节，听不清就放慢，再也不怕难句；  
- 🎙 **跟读录音**：一键录音影子跟读，对比原声，找出发音差距。
